# BaZi Prompt Kit

给没有用过 skill 的人准备的可复制 prompt 套件。

## 文件说明

- `full-prompt.md`
  适合 ChatGPT、DeepSeek、Kimi、豆包里相对更能吃长 prompt 的模型。
- `lite-prompt.md`
  适合免费模型、短上下文模型，规则更精简。
- `two-step-prompts.md`
  适合一次发太长效果不好时，先做“解读底稿”，再做正式解读。
- `user-template.txt`
  给普通用户直接填写出生信息或四柱。

## 建议使用顺序

1. 优先试 `full-prompt.md`
2. 如果输出还是散、短、跑偏，改试 `lite-prompt.md`
3. 如果模型容易忘规则，改用 `two-step-prompts.md`

## 最实用建议

- 如果用户已经有四柱，优先直接给四柱，效果通常比只给生日更稳定。
- 如果模型不会精确排盘，不要逼它装作精确。让它先按用户给的四柱解读。
- 复制 prompt 时，尽量把“我的信息如下”那部分一起发出去。
