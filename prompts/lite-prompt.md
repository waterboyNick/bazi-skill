# Lite Prompt

适合免费模型、短上下文模型。比完整版短，但保留最关键的约束。

```text
请用现代中文做一版“讲透的人话八字解读”。

要求：
- 保留八字、四柱、大运、流年的框架
- 不要玄学腔，不要故弄玄虚，不要只讲半句
- 不要只分析性格，优先讲命势、过去应事、未来几年、接下来一年、感情事业财运
- 如果我已经给了四柱，就按四柱解读
- 如果我只给出生信息，而你无法可靠精确排盘，就直接说明不确定，不要装作精确
- 不要默认贴来源链接

你的输出顺序：

1. 一句话总断
2. 整体命势
3. 大运主线
4. 过去已经应过的事（按强弱排序）
5. 未来三到五年逐年讲
6. 接下来一年按时间段讲
7. 感情、事业、钱分别给结论

每个年份或阶段都必须讲清楚：
1. 最强结论
2. 主要领域
3. 现实里像什么事
4. 为什么这么看

每个术语第一次出现，都翻译成人话：
- 日主 = 这个人的核心运作方式
- 官杀 = 压力、规则、权威、竞争
- 印 = 学习、证书、恢复、体系支持
- 食伤 = 表达、输出、观点、创意
- 财 = 钱、资源、感情/伴侣主题
- 比劫 = 自主性、同辈竞争、按自己方式来

语气要求：
- 像真的会看事的人
- 但讲得现代、清楚、具体
- 可以说高把握 / 中把握 / 偏推测
- 不要在结尾说“如果你愿意我还能继续讲”

我的信息如下：

出生信息：
- 公历生日：
- 出生时间：
- 出生地：
- 性别：

如果已有四柱：
- 四柱：

我最想看：
```
