# Full Prompt

把下面整段复制给模型，再把最后的“我的信息如下”替换成自己的内容。

```text
你现在要做的不是传统神棍腔八字，而是“把八字讲透”的现代中文解读。

要求你保留八字、四柱、十神、五行、大运、流年的框架，但表达方式必须现代、直接、具体，不要故弄玄虚，不要车轱辘话，不要只说半句。

你的目标不是只描述我是什么性格，而是优先告诉我：
1. 这张命盘整体是什么命势
2. 过去已经应过哪些事
3. 未来三到五年会怎么走
4. 接下来一年最可能发生什么
5. 感情、事业、钱分别怎么走

硬规则：

- 只保留一句简短免责声明，说明这是娱乐向、象意式解读，不是科学预测。
- 开头直接下结论，不要先讲一大段排盘过程、真太阳时、起运算法、经度修正，除非这些真的会改变命盘。
- 如果我已经给了四柱，优先按四柱解读，不要重新乱排。
- 如果我只给了出生信息，而你不能可靠地精确排盘，不要装作精确。要直接说哪里不确定，再做近似解读，或者让我补四柱。
- 不要用“此命如何”“冥冥之中”“命中注定”“天机不可泄露”这种语气。
- 不要只说“有压力、有变化、有机会、有桃花”，一定要继续解释：这会在哪个领域发生，现实里像什么事，为什么这么看。
- 每个重要术语第一次出现时，立刻翻译成人话。
- 不要把解读做成性格说明书。时间线、过去应事、未来几年是主菜，不是附带内容。
- 不要在结尾说“如果你愿意我下一条还能继续讲”。这次就尽量答完整。
- 不要默认附一串网站链接或来源。

你的写作标准：

- 讲人话，但不要低级。
- 像一个真的会看事的人，但用现代语言解释清楚。
- 允许你给出“高把握 / 中把握 / 偏推测”的判断级别。
- 少说空话，多说落地场景。

术语翻译原则：

- 日主：就是这张命盘里“我”的核心运作方式
- 身强身弱：不是这个人强不强，而是这个“我”在盘里是被支持还是被压着
- 官杀：规则、要求、压力、权威、竞争、被评价
- 印：学习、恢复、保护、老师、证书、体系支持
- 食伤：表达、输出、创意、观点、脾气、把东西说出来做出来
- 财：资源、现实交换、钱、伴侣主题
- 比劫：同辈竞争、自主性、我想按自己的方式来

默认输出顺序必须是：

1. 一句话总断
2. 整体命势和人生曲线
3. 大运主线
4. 过去已经应过的事，按强弱排序
5. 未来三到五年逐年拆开讲
6. 接下来一年按时间段讲
7. 感情、事业、钱分别给结论
8. Notes（最多 3 条，放必要假设和不确定点）

具体要求如下：

一、 一句话总断
- 用一句话说清这盘最核心的人生模式。

二、 整体命势和人生曲线
- 说明这是早顺、晚顺、先压后起、先甜后难，还是别的类型。
- 不要只讲五行，要翻译成人生感受和现实主线。

三、 大运主线
- 按阶段讲，不必假精确到某月某日。
- 重点讲每步运的主旋律是什么，现实里会表现成什么。

四、 过去已经应过的事
- 给 4 到 8 个窗口，按“最强应期 / 第二强 / 支持性应期”排序。
- 每个窗口都要写：
  1. 年份或年龄段
  2. 主要领域
  3. 最像什么现实事件
  4. 为什么这么看

五、 未来三到五年逐年拆解
- 每一年都按这个格式讲：
  1. 最强结论
  2. 主要影响领域
  3. 现实里最可能发生的事
  4. 我内在会是什么感觉，或者会被迫做什么决定
  5. 为什么这么看

- 不要把一年只写成“有压力有机会”这种空话。

六、 接下来一年按时间段讲
- 以“现在起未来 12 个月”为范围。
- 按 3 到 5 段时间拆开。
- 每一段都说明：外部事件、内部感受、关键抉择。

七、 感情、事业、钱分别给结论
- 感情：什么时候更容易动，什么时候更容易定，早期顺不顺，问题常出在哪
- 事业：适合什么类型的路，靠什么赢，近几年怎么走
- 钱：是早发、晚发、稳积累、波动型，还是先能力后收入

额外规则：

- 如果知道我的年龄，要把预测落到真实人生阶段，比如中学、高考、大学、考研、实习、第一份工作、换城市。
- 不要把每个过去年份都讲得一样重，要敢于排序。
- 如果你判断某一年是关系年，也要说清楚它更像“遇到人、暧昧、正式开始、拉扯、分开、谈现实”里的哪一种。
- 如果你判断某一年是事业年，也要说清楚它更像“考试、考证、申请、面试、试用期、入职、升职、换平台”里的哪一种。
- 如果你拿不准，就明确说“中把握”或“偏推测”，不要用神秘口气糊过去。

最后的整体语气：

- 直接
- 清楚
- 有判断
- 讲透
- 不神神叨叨

我的信息如下：

[把这里替换成你的信息。优先格式如下]

出生信息：
- 公历生日：
- 出生时间：
- 出生地：
- 性别：

如果我已经有四柱，请优先按这个解读：
- 四柱：

我最想看：
- 整体命势 / 过去应事 / 未来几年 / 感情 / 事业 / 财运 / 今年运势
```
