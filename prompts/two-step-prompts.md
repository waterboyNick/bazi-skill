# Two-Step Prompts

如果单条 prompt 太长、模型容易忘规则，可以拆成两步。

## 第一步：只生成解读底稿

先把下面这段发给模型。  
这一步不要让它正式开讲，只让它整理命盘假设、时间线框架和待解读重点。

```text
你先不要正式解读，只做“八字解读底稿”。

任务：
- 如果我给了四柱，就以四柱为准
- 如果我只给了出生信息，而你不能可靠精确排盘，就明确写出不确定点，不要装作精确
- 用尽量短的文字输出一个“解读底稿”，供下一步正式解读

输出格式固定如下：

1. 采用的命盘
- 四柱：
- 如果有不确定点，写清楚：

2. 用户当前人生阶段
- 年龄：
- 更像处在：中学 / 高考 / 大学 / 实习 / 第一份工作 / 其他

3. 盘面主轴
- 日主：
- 最强元素或主题：
- 更像：先压后起 / 晚起 / 变化多 / 稳中慢升 / 其他

4. 最值得看的时间线
- 过去最强 3 个窗口：
- 未来最值得看 3 到 5 年：
- 接下来一年主线：

5. 重点问题
- 感情：
- 事业/学业：
- 钱：

规则：
- 不要正式展开分析
- 不要写大段命理解释
- 不要贴来源链接
- 只生成底稿

我的信息如下：

出生信息：
- 公历生日：
- 出生时间：
- 出生地：
- 性别：

如果已有四柱：
- 四柱：
```

## 第二步：根据底稿做正式解读

把第一步模型输出的“底稿”复制回来，再加上下面这段发出去。

```text
现在请根据上面的“八字解读底稿”，输出正式解读。

要求：
- 用现代中文
- 不要玄学腔
- 不是只讲性格，要优先讲时间线和断事
- 每个判断都要讲清楚现实里像什么、为什么这么看
- 不要贴来源链接
- 不要在结尾说“如果你愿意我还能继续讲”

输出顺序固定如下：

1. 一句话总断
2. 整体命势和人生曲线
3. 大运主线
4. 过去已经应过的事（按强弱排序）
5. 未来三到五年逐年拆解
6. 接下来一年按时间段讲
7. 感情、事业、钱分别给结论
8. Notes（最多 3 条）

每个年份或阶段都要按这个格式：
1. 最强结论
2. 主要影响领域
3. 现实里最可能发生的事
4. 我内在会是什么感觉，或者会被迫做什么决定
5. 为什么这么看

术语第一次出现必须翻成人话：
- 日主 = 核心运作方式
- 官杀 = 压力、规则、权威、竞争
- 印 = 学习、证书、恢复、体系支持
- 食伤 = 表达、输出、观点、创意
- 财 = 钱、资源、伴侣主题
- 比劫 = 自主性、同辈竞争

结尾要有一个短的底线判断，总结这盘最核心的人生走向。
```
